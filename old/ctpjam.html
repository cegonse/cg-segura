Capture the Pixel (LowRezJam '16)

Capture the Pixel is a multiplayer shooter arena game, where the red and blue team fight to steal the pixel to to the other team. The team which keeps the other team's pixel when the round finishes wins.

The game features several weapons and fast paced action. Sadly the game requires the server side to run and it was closed few weeks after the jam as nobody was playing the game.
